# Configuration Parameters

- **input_file**: Path to the input file
- **make_fixed_size**: Flag to indicate if the output image should have a fixed size
- **img_width**: Width of the output image
- **img_height**: Height of the output image
- **min_tps_to_cluster**: Minimum number of time points to form a cluster
- **drift_direction**: Direction of the drift
- **save_img_dataset**: Flag to indicate if the image dataset should be saved
- **save_label_process**: Flag to indicate if the label for the process should be saved
- **save_label_direction**: Flag to indicate if the label for the direction should be saved
- **x_margin**: Margin in the x-direction for the output image
- **y_margin**: Margin in the y-direction for the output image
- **only_collection**: Flag to indicate if only the collection plane should be used
